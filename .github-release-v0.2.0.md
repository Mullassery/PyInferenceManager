## 💻 Quick Example

```python
from pyinferencemanager import Orchestrator

# Minimal setup
orch = Orchestrator(mode="local_first")

# Execute with automatic routing
result = orch.run(
    task="analyze_document",
    file="contract.pdf",
    privacy="low"  # Allow cloud fallback
)

print(f"Output: {result.output}")
print(f"Cost: ${result.total_cost_usd:.4f}")
print(f"Latency: {result.total_latency_ms}ms")
print(f"Cache hits: {result.cache_hits}")
```

## 🔒 Security & Privacy

- No telemetry collection (self-hosted observability)
- Privacy-first: Can force local execution with `privacy="high"`
- No model weights in binary (external APIs only)
- Support for environment-variable API keys
